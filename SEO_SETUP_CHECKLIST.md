# SEO Setup Checklist & Next Steps

## 🚀 Immediate Actions (Next 24 Hours)

### 1. **Google Search Console Setup**
```
1. Go to https://search.google.com/search-console
2. Click "Add property" → Select "URL prefix"
3. Enter: https://fushiguro.services
4. Verify ownership via:
   - DNS record (recommended)
   - HTML file upload
   - HTML tag in head
5. Go to Sitemaps → Add new sitemap
6. Submit: https://fushiguro.services/sitemap.xml
```

### 2. **Google Analytics Setup**
```
1. Go to https://analytics.google.com
2. Create new property for "Fushiguro"
3. Get your Measurement ID (format: G-XXXXXXXXXX)
4. In src/pages/index.astro, replace YOUR_GA_ID with your ID
5. Uncomment the gtag lines
```
